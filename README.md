MPU6050-WebPlot
===============

plotting the signal from MPU6050

Latest version: v0.2

Preview
-------

![web plot](preview.png)

Environment
-----------

- OS X 10.9
- Python 2.7.6

Requirement
-----------

**Python**

- pyserial 2.7
- tornado 3.1.1

**Javascript**

- jQuery 2.0.3
- Flot 0.8.2
- Semantic UI 0.11.0

**v0.1.1**

**v0.1**

	- Z-axis acceleration
	- Roll angle velocity
	- Pitch angle velocity
